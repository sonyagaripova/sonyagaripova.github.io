A map of landscape visibility of national parks

The aim of the project was to develop a map of background seismicity domains that will be used as input data in General Seismic Regionalizing (GSR) map development. It implies processing of .... in domains, fault zones and epicenters of seismic events separately. So the task was to distinguish such domains.
My colleagues decided that background seismicity depends on a few parameters:

- Earth’s gravity anomaly;
- heat flow;
- topography range;
- active faults;
- seismic events with magnitude from 3 to 5.5

Input data

Входные данные - это global digital elevation model SRTM 1-arc sec and satellite image Landsat 8

5) The map of seismic events density with magnitude from 3 to 5.5, created by Point density tool. Events with higher magnitude should be considered in other processing

After obtaining and analysing maps, I used Iso Cluster Unsupervised Classification script, varying
- the number of classes;
- minimum class size;
- and sample interval.

The script takes raster data that is above mentioned maps. Each resulting class represents a territory with homogenius pattern.

As we can see, plain territories such as Russian Plain and West Siberian Plain, are represented mostly by two classes differed due to gravity anomaly, while mountain areas such as Urals, Altai, Sayan, Caucasus, Cherskiy Range and all of the Far East close to Pasific Belt are more speckled due to consisting of more classes.
